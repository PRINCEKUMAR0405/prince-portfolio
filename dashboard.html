Google Analytics Integration

Setup Instructions

  1. Create a Google Analytics 4 property at analytics.google.com
  2. Get your Measurement ID (format: G-XXXXXXXXXX)
  3. Replace 'G-XXXXXXXXXX' in both index.html and dashboard.html with your actual ID
  4. Deploy your website and wait 24-48 hours for data to appear in Google Analytics
  5. Access detailed reports including real-time traffic, demographics, and user behavior

What Google Analytics Tracks:

  • Real-time active users
  • Geographic location of visitors
  • User demographics and interests
  • Page views and session duration
  • Traffic sources and campaigns
  • Device and browser information
  • User flow and behavior patterns

About This Dashboard

This dashboard uses localStorage to track basic visitor information on the client side. It's limited to the last 100 visitors and data stored in your browser. For comprehensive analytics, including cross-device tracking, conversion metrics, and advanced reporting, use Google Analytics.
